## 1. OVERVIEW

SmartLED Controller is designed for controlling LED lighting devices via Bluetooth connectivity. We are committed to protecting your privacy and being transparent about our data practices.

### Key Points:

- Most data is stored locally on your device

## 5. DATA STORAGE AND SECURITY

### 5.1 Local Storage

**Primary Storage Location:**
- Most data is stored locally on your device
- Data is protected by your device's security features
- We use industry-standard encryption for sensitive data
- You control when data is deleted

### 5.2 Cloud Storage (Optional)

**If You Enable Cloud Sync:**
- Data is encrypted in transit (HTTPS/TLS)
- Data is encrypted at rest on servers
- You can disable cloud sync at any time
- You can delete cloud data through app settings

### 5.3 Security Measures

**Technical Safeguards:**
- Encryption of sensitive data
- Secure communication protocols (HTTPS/TLS)
- Regular security updates
- Access controls and authentication

**Limitations:**
- No system is 100% secure
- You are responsible for device security
- Use strong device passwords/biometrics
- Keep your device and app updated

### 5.4 Bluetooth Security

- Bluetooth communications are local
- Pairing requires physical proximity
- We recommend secure pairing methods
- Disconnect when not in use
